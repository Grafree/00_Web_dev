Un document HTML peut être conçu et s'adapter à plusieurs périphérique. Pour cela il suffit de créer une feuille de style pour chaque support et ainsi redéfinir l'apparence du contenu selon l'espace et les dimensions de ceux-ci.

La liaison d'un fichier CSS s'établit par la balise link qui permet de définir le support pour chaque type de support grâce à l'attribut media. Ainsi, il peut y avoir une feuille de style pour l'affichage dans un navigateur (screen), pour l'impression (print) ou pour les support mobiles, tels que les téléphones mobiles (handheld)

Pour un navigateur, la feuille de style est définie par <link rel="stylesheet" type="text/css" href="css/site.css" media="screen" />. Les dimensions peuvent être fixes (px) ou relatives (%). La dimension des images peuvent être également redéfinie en fixe (px) ou relative (%).

Pour l'impression, la feuille de style est définie par <link rel="stylesheet" type="text/css" href="css/print.css" media="print" />. Les dimensions peuvent être fixes (pt, cm ou mm) ou relatives (%). La dimension des images peuvent être également redéfinie en fixe (cm ou mm) ou relative (%).

Pour les supports portables, la feuille de style est définie par <link rel="stylesheet" type="text/css" href="css/portable.css" media="handheld" />. Les dimensions peuvent être fixes (px) ou relatives (%). La dimension des images peuvent être également redéfinie en fixe (px) ou relative (%).
