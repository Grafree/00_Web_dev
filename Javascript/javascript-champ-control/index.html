Champ simple

Un champ s'accompagne d'une étiquette identifiée par la balise <label>. Elle contribue sur le plan sémantique a lier le texte à l'élément du formulaire auquel il se rattache.

Un texte explicatif sous le champ contribue à clarifier ce qui est attendu comme information dans l'élément du formulaire. Sa position contribue à

L'icône d'aide, complète les informations sur l'utilisation de l'information indiquée dans l'élément du formulaire. L'attribut title permet d'ajouter ces informations afin qu'elles soient visibles lors du survol du curseur sur l'icône.

L'attribut placeholder (apparue avec la version HTML 5) permet de citer un exemple dans un champ. Ce qui n'était possible auparavant qu'à l'aide de javascript.
